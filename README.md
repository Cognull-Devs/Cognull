# Cognull - Plataforma Institucional

> Arquitetando inteligência digital em escala

## Sobre o projeto

Landing page institucional da Cognull com foco em posicionamento técnico, apresentação da equipe fundadora e abertura de contato direto via WhatsApp.

No momento, o projeto não apresenta portfólio publicado de cases, e a navegação foi ajustada para refletir esse estágio.

## Funcionalidades

- Visual moderno com identidade tecnológica e sóbria
- Layout responsivo para desktop e mobile
- Navegação suave por âncoras
- Seções institucionais: soluções, arquitetura, pipeline, equipe e matriz tecnológica
- Modal de contato com a equipe (WhatsApp)
- Links sociais de LinkedIn e GitHub por membro da equipe

## Tecnologias

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query
- Lucide React

## Como rodar o projeto

Pré-requisitos:

- Node.js 18 ou superior
- npm

Passos:

1. Clone o repositório.
2. Entre na pasta do projeto.
3. Instale as dependências com npm install.
4. Inicie em desenvolvimento com npm run dev.

Aplicação local:

- http://localhost:5173

## Scripts

- npm run dev: inicia ambiente de desenvolvimento
- npm run build: gera build de produção
- npm run preview: serve o build localmente
- npm run lint: executa lint
